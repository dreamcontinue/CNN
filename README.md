# CNNs for image classification
Backup of code I used to train CNNs for image classification from scratch.

### Requirement
- `pytorch 1.1.0+`
- `torchvision`
- `numpy`
- `pyyaml`
- `tqdm`
- `tensorboardX`
- `pillow`

### Dataset
- `CIFAR-10`
- `CIFAR-100`
- `ImageNet(2012)`

### Usage
run `train.py`, `validata.py`, `test.py`
